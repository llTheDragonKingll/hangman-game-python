# 🎯 Hangman Game (Python)

A command-line Hangman game built using Python where the player guesses letters to reveal a hidden word.

---

## 🚀 Features

* 🔤 Random word selection
* ❤️ Lives system
* 🧠 Letter-by-letter guessing
* 🎮 ASCII-based hangman visuals
* 🔁 Continuous gameplay until win or loss

---

## 🛠️ Tech Stack

* Python

---

## 📂 Project Structure

```text
hangman-game-python/
│
└── main.py
```

---

## ⚙️ How to Run

```bash
python main.py
```

---

## 💡 Notes

* Tracks guessed letters and updates word dynamically
* Reduces lives on incorrect guesses
* Displays hangman stages visually using ASCII art
